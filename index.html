Selected Work

Selected Projects

Modeling Human Action Sequences

Scalar-on-process modeling, sparse learning, behavioral event logs

Built a model for variable-length action logs using B-spline bases and adaptive Group LASSO. The analysis modeled 19,346 action events from 2,685 participants and improved repeated-holdout mean test accuracy from 93.22% to 96.20% over a frequency-only baseline.

Latent Structure in Sequential Behavioral Data

Latent variables, simulation design, sparse selection, reproducible evaluation

Developed a Python research pipeline for latent-state sequence models, weak supervision, sparse regularization, refitting, and validation on simulated and real behavioral data. The work emphasizes method comparison, identifiability, uncertainty, and failure-mode analysis.

Climate Signals and Extinction-Event Modeling

Scientific time series, proxy alignment, regularized prediction

Aligned environmental and geochemical proxy measurements to a shared time scale, engineered lag and rolling-window features, and compared LASSO, Elastic Net, Random Forest, XGBoost, and SVR models for interpretable early-warning analysis.
